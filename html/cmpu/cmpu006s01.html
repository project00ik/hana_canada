송금 보내기

  • 온라인뱅킹

    온라인뱅킹 서비스 내 "해외송금" 기능을 이용하여 CAD 10,000 상당액까지 보내실 수 있습니다.

  • 영업점 방문

    가까운 영업점에 직접 방문하셔서 송금 보내기도 가능합니다.

  • 송금통화

    캐나다 달러, 미국 달러, 한국 원화, 유로*, 일본 엔화*, 중국 위안화*로 송금을 보내실 수 있습니다.
    * 지점 방문을 통해 송금 가능

  • 송금 보낼 시 필요한 정보

    • 은행명
    • 은행코드(SWIFT Code, Transit No.)
    • 계좌번호
    • 성함
    • 주소
    • 전화번호
    • 미국 송금 시 : ABA Routing #
    • 유럽 송금 시 : IBAN

송금 받기

  • 송금 받을 시 제공 정보

    • SWIFT BIC Code : KOEXCATT
    • Institution No. (은행코드) : 275(또는 0275)
    • Transit No. (지점 번호)
    • 수취인의 캐나다하나은행 계좌번호와 이름
